# Crowd sprite (optional local copy)

The hero animation uses Open Peeps (CC0). By default it loads:

`https://s3-us-west-2.amazonaws.com/s.cdpn.io/175711/open-peeps-sheet.png`

To serve locally, save that file as `open-peeps-sheet.png` in this folder and set `src="/images/peeps/open-peeps-sheet.png"` in `LandingPage.jsx`.

Grid: **15 rows × 7 cols** (must match the sprite sheet layout).
